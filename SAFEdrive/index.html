An 8+ Exabyte D: Drive

Everyone thinks the SAFE Network is just a huge decentralized Dropbox. That's not true. It's the foundation for a whole new, encrypted, p2p internet for freedom. Dropbox will just be one of the countless applications built on top of it.

History

Unlike MegaUpload, Google Drive, Mediafire, or any other vulnerable, server-based file hosting service of today, the SAFE Network ensures every single file will safely exist online forever, by backing up all data across every online computer in the world. This will be facilitated by an app like SAFEdrive, which has not yet been developed. This app is central to how this new internet will function, therefore the person who codes it will be earning countless SafeCoin around the clock, as everyone who uses the internet will be using this app, to an extent. First mentioned on the forums.

Features

The ability to upload public and private files to the SAFE Network in exchange for the equivalent amount of SafeCoins, and organize them to your liking, will be the main function of this app. This will be data's new main portal for getting onto the internet, and almost every file in the world--video, music, documents, 3d models, OS's, anything--will use this SAFEdrive code to enter into the interwebs. Needless to say, this will be a very simple, but incredibly popular--and therefore lucrative--app to build... So make it!

Launch date

It was expected that the network would launch at the end of last year, (and this app soon after,) however an amazing new coding language came out called "Rust" that turned out to be perfect for the SAFE Network to be created with. So, the team decided to restart the whole network from scratch in Rust. This means we will have to wait a few more months to see the network or this app. Follow it on Github!
